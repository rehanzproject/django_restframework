lunak yang di buat untuk memandu desain dan pengembangan dari arsitektur Dunia Web Luas
API dibuat untuk mengijinkan klien untuk berbicara dengan server , dengan maksud transfer dari klien ke server menggunakan HTTP request

Kelebihannya lebih mudah untuk menyajikan data, karena headless architectur tidak perlu front-end untuk menyajikan data
contoh sederhana jika kita menyajikan dengan Front-end menggunakan Fetch method (Folder exampleFetch).
